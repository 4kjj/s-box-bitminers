# s-box-bitminers
DXRP Bitminer Addon
A fully modular, proximity-powered crypto mining economy addon built for the DXRP gamemode in s&box. Utilizing the modern Source 2 Scene System, this addon replaces clunky old-school cable mechanics with a clean, performant, wireless proximity network and an interactive 3D world UI.
Features
Wireless Proximity Power: Power sources (batteries) scan a 5-foot radius to automatically distribute power to nearby shelves. No physical cables required, reducing physics engine overhead and lag.
Modular Upgrades: Players spawn a base shelf and can interact with the integrated screen to purchase hardware upgrades, instantly raising their hash rate and cash generation.
Interactive 3D World UI: Built natively using s&box Razor panels. Displays real-time power connection status, revenue generation, and responsive upgrade/withdrawal buttons.
Fuel & Overclocking Mechanics: Batteries require paid gas refills to stay operational. By default, a battery powers up to 2 shelves (lasting 1 hour for one shelf, 45 minutes for two). Players can pay a premium to overclock the system, allowing it to support a 3rd shelf for a dense 40-minute mining cycle.
Secure PIN Locking: Protect hardware investments from thieves. Players can toggle a local 4-digit PIN lock directly from the screen, blocking unauthorized wallet withdrawals or upgrades.
Server-Side Job Protection: Deeply integrated logic ensures only players with the official "bitminer" job role can spawn or deploy the equipment.
